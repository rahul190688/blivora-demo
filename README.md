# Apex Analytix Ticketing System

Custom ticketing solution built on Frappe Helpdesk for Apex Analytix.

## Features

- Based on Frappe Helpdesk
- All Freshdesk features
- Custom enhancements for Apex Analytix

## Installation

```bash
bench get-app https://github.com/rahul190688/blivora-demo.git
bench --site [site-name] install-app blivora_demo
```
